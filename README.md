# HashBrown CMS
A free and open-source CMS built with Node.js and MongoDB

## Building and running HashBrown from source
```
git clone https://github.com/Putaitu/hashbrown-cms.git
cd ./hashbrown-cms
npm install
node hashbrown.js
```
